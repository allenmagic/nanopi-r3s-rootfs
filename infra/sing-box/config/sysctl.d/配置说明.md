# sysctl 配置说明

本目录下的 `.conf` 文件用于调整路由器内核参数，涵盖性能调优与转发安全控制。

---

## 1. `80-performance.conf`

针对 2G 内存与高并发连接的性能优化：

- 连接跟踪容量：`nf_conntrack_max=200000`，`nf_conntrack_buckets=50000`。
- 连接超时优化：缩短 `SYN_RECV` 与 UDP 超时，降低占用。
- 缓冲区与队列：提升 `rmem/wmem` 与 `somaxconn`、`netdev_max_backlog`。
- 抗 SYN flood：`tcp_syncookies=1`。
- 拥塞控制：`fq_codel` + `bbr`。
- 系统缓存：`swappiness=10`、`vfs_cache_pressure=50`。

---

## 2. `90-router.conf`

路由转发与基础安全设置：

- 开启 IPv4/IPv6 转发：`ip_forward=1`、`ipv6.conf.all.forwarding=1`。
- 关闭 `rp_filter`：便于 TProxy/多出口场景（否则可能丢包）。
- 禁止 ICMP Redirect 与源路由：`accept_redirects=0`、`send_redirects=0`、`accept_source_route=0`。

---

## 3. 调整建议

- 修改连接数或内存配置时，同步调整 `nf_conntrack_*` 与缓冲区参数。
- 启用 `rp_filter` 前需确认是否使用 TProxy/VPN 路由，以免回包被丢弃。
- 修改后应用：`sysctl -p /etc/sysctl.d/*.conf`。
